# News Scraper (Python)

A Python-based web scraping project that extracts clean and structured news articles from websites.

## Overview
This project focuses on extracting meaningful article data while removing ads and irrelevant content.  
It supports both static and dynamic web pages.

## Features
- Extracts article title, publish date, content, and images
- Handles dynamic pages using Selenium
- Cleans unnecessary ads and noise
- Outputs structured data (JSON)

## Tech Stack
- Python
- BeautifulSoup
- Selenium

## Use Cases
- News aggregation
- Backend data pipelines
- Content analysis and automation

## How to Run
```bash
pip install -r requirements.txt
python scraper.py
